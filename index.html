Customer Churn Prediction

Overview: This project focuses on building a machine learning model to predict customer churn for a telecom company. Identifying at-risk customers early allows for targeted retention strategies.

Methodology

We used Python with **Pandas** for data cleaning, **Scikit-learn** for model training, and **Matplotlib** for data visualization. Key steps included handling imbalanced data, feature engineering, and evaluating several classification algorithms.

Model Performance

Model Accuracy F1-Score ROC-AUC
Logistic Regression 85% 0.78 0.82
Random Forest 91% 0.89 0.91
Gradient Boosting 90% 0.88 0.90

Final Output

The final Random Forest model provided the best performance. The output was an interactive dashboard to visualize churn probability and key factors influencing churn.

Housing Price Regression

Overview: Developed a regression model to forecast housing prices using data from various features like square footage, number of rooms, and location.

Methodology

The project involved extensive data visualization with **Seaborn** to understand correlations. We used a **Linear Regression** model as a baseline and then a **Gradient Boosting Regressor** to improve accuracy, handling categorical variables with one-hot encoding.

Model Performance

Model R-squared RMSE
Linear Regression 0.75 45,000
Gradient Boosting 0.92 22,000

Key Findings

The model identified that the total square footage and location were the most significant predictors of a house's price.

E-commerce Sales Analysis

Overview: Conducted an extensive exploratory data analysis (EDA) to uncover sales trends, customer behavior, and product performance for an online retail store. This project demonstrated proficiency in data manipulation and visualization.

Methodology

The project used **Pandas** for data cleaning and aggregation, and **Matplotlib** with **Seaborn** for visualizations. We performed time-series analysis to find seasonal trends and used a pivot table to find the top-selling products.

Key Insights

  • Sales peaked during the holiday season (November-December).
  • The top three best-selling products accounted for 25% of total revenue.
  • Customers who buy Product A are highly likely to also buy Product B.

Final Report

The insights were compiled into a comprehensive report and a presentation, highlighting actionable recommendations for the marketing team.

Sentiment Analysis of Tweets

Overview: Used NLP techniques to classify the sentiment of a dataset of tweets as positive, negative, or neutral. This project highlights skills in text preprocessing and machine learning with text data.

Methodology

The project used **NLTK** for tokenization and stop-word removal. A **CountVectorizer** was used to convert text data into numerical features, which were then fed into a **Logistic Regression** model for classification.

Model Performance

Metric Score
Accuracy 0.89
F1-Score (Macro) 0.87

Visualization

A word cloud was generated to visualize the most frequent words in positive and negative tweets, providing a quick summary of the sentiment.
